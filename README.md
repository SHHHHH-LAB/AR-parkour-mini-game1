·	项目简介：基于Unity制作移动端AR跑酷小游戏，支持多角色切换，依靠手机触控操控角色移动、跳跃躲避障碍物，搭载计时计分系统，适配移动端操作。
·	技术实现：
·	1. 角色管理：通过CharacterChange脚本完成角色选择，使用PlayerPrefs保存角色选择记录；CharacterLoader加载对应角色，搭配动画重写控制器实现各角色独立动画。
·	2. 玩家运动系统：借助Rigidbody物理组件编写角色控制脚本，实现三车道切换、跳跃落地逻辑，增加悬空容错判定；碰撞触发障碍物扣分。
·	3. 移动端交互：CharacterCamera实现单指旋转视角、双指缩放镜头；UI按钮绑定左右移动、跳跃操作。
·	4. 游戏与场景控制：单例UIController统一管理倒计时、分数、开局与游戏结束；障碍物、场景道具通过脚本实现循环移动、自转、往复平移效果。
·	5. 低耦合模块化开发，运动速度、时长、车道距离等参数可在面板直接修改，无需改动代码。
项目成果：完整实现角色选择、跑酷闯关、计分计时等核心玩法，适配手机多点触控；项目逻辑稳定，代码分层清晰，已打包移动端安装包并上传GitHub归档。
